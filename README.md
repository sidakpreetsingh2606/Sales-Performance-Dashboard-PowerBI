# Sales Performance Dashboard - Power BI

## 📊 Project Overview
This Power BI dashboard analyzes sales performance using the Superstore dataset.

The dashboard provides insights into:
- Sales trends over time
- Category-wise sales performance
- Profitability by sub-category
- KPI metrics like Sales, Orders, and Profit
- Interactive filtering by Year, Region, and Category

---

## 🛠 Tools Used
- Power BI
- Data Visualization
- Data Cleaning
- KPI Analysis

---

## 📌 Key Insights
- Technology category generated highest sales
- Sales increased during September–November
- Phones and Copiers showed strong profitability
